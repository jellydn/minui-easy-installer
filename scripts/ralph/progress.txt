## Codebase Patterns
- Tauri v2 with React frontend, Vite bundler

- Rust edition 2021, tauri 2.x
- Icons must be RGBA PNG format (not RGB)
- Pin `plist` to 1.7.0 and `serde_with` to 3.11.0 to avoid `time` crate v0.3.47+ conflicts with `cookie` 0.18.1
- Use `oxlint` for linting (already configured in package.json)
- Frontend dev server on port 1420
- Use `vitest` for TypeScript unit testing (tests in `*.test.ts` files)
- Device profiles and types live in `src/types/`
